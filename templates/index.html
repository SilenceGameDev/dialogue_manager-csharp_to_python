Why I made this project

I have been coding c# and making games in unity for 6 years. I have been making educational games for Legends Of Learning for 5 years. I wanted to make something small so I can get accustomed to the differences between python and c#.

Here is the Dialogue Manager flow in Unity


The dialogue manager handles playing the correct dialogue based on the parameters passed in. DialogueData is used to quickly set up a sequence of dialogue. DialogueActivator can be an object in the world that will start the dialogue when the player triggers it.


Please excuse the placeholder art. This game hasn't been released yet ;)

Inspector View

There is one DialogueManager object that exists in the world. It handles bringing up the dialogue widow and playing the correct dialogue.

Player Triggering Dialogue

Here in the Scene View you can see the player walking into the dialogue trigger (the green box).

Going through Dialogue

After the player triggers the dialogue they click the continue button to go on to the next sentence. When there are no more sentences, the dialogue window closes.
