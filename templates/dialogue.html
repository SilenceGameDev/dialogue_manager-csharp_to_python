How it works

Enter a letter from A-H and an Index below. Click the "Get Dialogue" button to return the dialogue at the selected index, if it exists. The page will reload to display the correct dialogue. All indexes start at 0. Plays a random dialogue if there isn't one active already.

The dialogue was separated by letters to match the section they were a part of. The index makes it easy to increment to the next sentence in the dialogue. It also makes testing easier.
